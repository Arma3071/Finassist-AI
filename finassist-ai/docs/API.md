# API Documentation

Base URL: `http://localhost:8000/api`. Full interactive docs (Swagger UI)
are auto-generated by FastAPI at `http://localhost:8000/docs`.

Authenticated endpoints accept an optional `Authorization: Bearer <token>`
header (obtained from `/auth/login`); it's used to attribute uploads and
chat history to a user but is not strictly required to call the endpoints.

---

## `POST /auth/register`

Register a new user.

**Body**
```json
{ "username": "alice", "password": "hunter2pass" }
```

**Response `200`**
```json
{ "status": "created", "username": "alice" }
```

**Response `409`** — username already exists.

---

## `POST /auth/login`

**Body**
```json
{ "username": "alice", "password": "hunter2pass" }
```

**Response `200`**
```json
{ "token": "a1b2c3...", "username": "alice" }
```

**Response `401`** — invalid credentials.

---

## `POST /upload`

Multipart file upload. Accepts `.pdf`, `.docx`, `.txt`, `.md`.

**Response `200`**
```json
{
  "document_id": "9f8e...",
  "filename": "sample_report.md",
  "chunks_created": 4,
  "status": "success",
  "message": ""
}
```

---

## `DELETE /documents`

**Body**
```json
{ "document_id": "9f8e..." }
```

**Response `200`**
```json
{ "document_id": "9f8e...", "chunks_deleted": 4 }
```

---

## `POST /chat`

**Body**
```json
{
  "session_id": "my-session-1",
  "message": "What was Q3 revenue growth?",
  "top_k": 5,
  "metadata_filter": null
}
```

**Response `200`**
```json
{
  "session_id": "my-session-1",
  "answer": "Revenue grew 14% year-over-year to $482 million [S1].",
  "confidence": 0.82,
  "sources": [
    {
      "chunk_id": "abc::0::12345678",
      "document_name": "sample_report.md",
      "text": "The company reported revenue of $482 million...",
      "score": 0.91,
      "metadata": {"filename": "sample_report.md"}
    }
  ],
  "tool_calls": [],
  "route": "retrieval",
  "latency_ms": 842.3,
  "timestamp": "2026-07-21T10:00:00Z"
}
```

`route` will be one of `retrieval`, `tool`, `direct`, `hybrid`.

## `POST /evaluate`

Runs the golden evaluation dataset (`backend/evaluation/dataset.py`)
through the agent and returns per-question and aggregate metrics.

**Response `200`**
```json
{
  "items_evaluated": 4,
  "results": [
    {
      "question": "What was Q3 revenue and how much did it grow year-over-year?",
      "retrieval_precision": 1.0,
      "retrieval_recall": 1.0,
      "latency_ms": 640.2,
      "hallucination_rate": 0.0,
      "context_relevance": 0.88,
      "answer_relevance": 0.79
    }
  ],
  "aggregate": {
    "avg_precision": 0.95,
    "avg_recall": 0.9,
    "avg_latency_ms": 610.4,
    "avg_hallucination_rate": 0.05,
    "avg_context_relevance": 0.85,
    "avg_answer_relevance": 0.77
  }
}
```
